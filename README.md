# Meetapp

Final ReactJS project developed at the GoStack bootcamp. Web version of the application for meetup appointments.

Techs:

- ReactJS
- React Hooks
- React Router
- Redux
- Redux Saga
- Redux Persist
- Reactotron
- Styled Components
- Toastify
- Yup
- Axios
- Prop Types
- date-fns
- ESLint
